Darkrai Guide

The Pitch-Black Pokemon

Darkrai is a child sized pokemon, characterized by it's white "hair" and red "cowl".
The following should help inform new trainers on how to deal with this pokemon.

Typage

Darkrai is a Dark type pokemon. As a solo type Pokemon, it is far simpler to figure out what it is weak and strong against
Dark Type Pokemon follow the following Strengths and Weaknesses

Offensive Description
Strong (x2) Ghost, Psychic
Weak (x.5) Dark, Fighting, Fairy
Defensive Description
Weak (x2) Fighting, Bug, Fairy
Resists (x.5) Ghost, Dark, Psychic

Darkrai is a Mythical pokemon known for causing nightmares in whatever pokemon or people are near. The pokemon has been known to fall on similar move sets, but varieties have been recorded. Examples are Here, Here, and Here.
Snarl and Shadow Ball are preferred moves for Darkrai, but it is able to learn a variety of other moves. These moves are not always dark type moves, so be sure to have a variety of types to help avoid any Knocked out Pokemon in battle. A common moveset is the following

  1. Snarl
  2. Sludge Bomb
  3. Shadow Ball
  4. Dark Pulse

However, these are not the only moves a Darkrai can learn. Some of the following have been seen in Darkrai

  • Feint Attack
  • Focus Blast
  • Dream Eater
  • Hypnosis
  • Haze
