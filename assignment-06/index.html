Typage

Dusknoir is a Ghost type pokemon. As a solo type Pokemon, it is far simpler to figure out what it is weak and strong against
Ghost Type Pokemon follow the following Strengths and Weaknesses

Offensive Description
Strong (x2) Ghost, Psychic
Weak (x.5) Dark
No effect (x0) Normal
Defensive Description
Weak (x2) Ghost, Dark
Resists (x.5) Poison, Bug
No effect (x0) Normal, Fighting

Wild Dusknoir Fighting styles can vary wildly between locations, but can be observed Here, Here, and Here.
Shadow Ball and Shadow Sneak are preferred moves for Dusknoir, but they are able to learn a variety of other moves. These moves are not always ghost type moves, so be sure to have a variety of types to help avoid any Knocked out Pokemon in battle. A common moveset is the following

  1. Astonish
  2. Hex
  3. Shadow Ball
  4. Dark Pulse

However, these are not the only moves a Dusknoir can learn. Some of the following have been seen in Wild Dusknoir

  • Ominous Wind
  • Psychic
  • Poltergeist
  • Frustration
  • Return

Be sure to talk to your local trainer to see what moveset fits your battle style best, or to see what moveset your pokemon can learn to best work around any wild Dusknoir.
